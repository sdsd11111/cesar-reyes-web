# Imágenes para la Promoción de Artes Vivas 2025

Este directorio contiene las imágenes utilizadas en la página de promoción especial para artesanos del Festival Artes Vivas Loja 2025.

## Imágenes Requeridas

1. **hero-artes-vivas.webp**
   - Tamaño recomendado: 1920x1080px
   - Descripción: Imagen de fondo para el primer slide mostrando artesanos trabajando en el Festival Artes Vivas Loja.
   - Palabras clave: artesanos, festival artes vivas, loja, artesanías, trabajo artesanal

2. **google-vs-facebook.webp**
   - Tamaño recomendado: 1920x1080px
   - Descripción: Imagen de fondo para el segundo slide mostrando una comparación visual entre Google y Facebook.
   - Palabras clave: google vs facebook, redes sociales, comparación, marketing digital

3. **testimonio-artesano.webp**
   - Tamaño recomendado: 1920x1080px
   - Descripción: Imagen de fondo para el tercer slide mostrando a un artesano trabajando o mostrando sus productos.
   - Palabras clave: artesano, testimonio, producto artesanal, artesano trabajando

## Instrucciones

1. Reemplazar los placeholders con imágenes reales en formato WebP
2. Asegurarse de que las imágenes tengan buena calidad y estén optimizadas para web
3. Mantener los nombres de archivo exactamente como están definidos
4. Verificar que las imágenes tengan los derechos de uso correspondientes

## Optimización

Todas las imágenes deben estar optimizadas para web para garantizar tiempos de carga rápidos. Se recomienda:
- Usar formato WebP para mejor compresión
- Redimensionar las imágenes a las dimensiones recomendadas
- Comprimir sin pérdida significativa de calidad

## Créditos

Incluir los créditos correspondientes para las imágenes utilizadas, especialmente si son de terceros.
